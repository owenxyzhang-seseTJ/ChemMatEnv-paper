# ChemMatEnv-paper 简介

**ChemMatEnv-paper** 是面向材料、化学、环境三大领域的顶刊论文全流程 AI 协作技能，遵循 Agent Skills 开放标准，可在 ZCode、Claude Code、Codex CLI 等 20+ 工具中通用。

核心哲学是 **「结构优先于措辞」**——先诊断论文类型逻辑和声明-证据映射，再逐句润色。覆盖：

- **故事架构**：从证据提取核心命题，生成候选故事线，锁定 8 字段 story package（核心问题、瓶颈、设计原则、主声明、证据映射、范围边界等），防写作中故事漂移。
- **实时期刊情报**：起草前搜索目标期刊 author guidelines（字数上限、摘要格式、图表限额），按 ACS、RSC、Wiley、Nature、Science、Cell Press 六大家族适配。
- **章节 Playbook 起草**：Abstract 六步、Introduction 四段漏斗、Results 证据阶梯、Discussion 解读+hedging 校准、Conclusion 三部分收尾。
- **文献引用搜索**：分段→解析声明→5 级保守评估→导出 ENW/RIS/RDF，期刊范围覆盖六大出版家族。
- **质量闸门分级**：Full gate（新起草/大改）含顶刊表达检索+diction 校准、去 AI 味+中→英修复、逻辑链审计+反向提纲；Light gate（小修）仅后两项，同 session 结果可复用。
- **编辑+三轮审稿模拟**：Handling Editor 预审 scope/novelty/gaps，A/B/C 三位 reviewer 分域评审（方法证据/领域新颖性/逻辑风格），生成 review_action_matrix。
- **LaTeX 排版诊断**：float 松散、宽扁图、搁浅标题、多面板、SI 密度六类修复。
- **中→英修复**：8 种中文论文常见模式修复，含材料化学专用表征归属和术语一致性规则。
