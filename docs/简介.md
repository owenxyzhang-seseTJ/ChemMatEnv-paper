# ChemMatEnv-paper 简介

**ChemMatEnv-paper** 是一个面向材料、化学、环境三大领域的顶刊论文全流程 AI 协作技能。它遵循 Agent Skills 开放标准，可在 ZCode、Claude Code、Codex CLI、OpenCode、Hermes、Gemini CLI、Cursor 等 20+ 工具中通用。

传统写作辅助只改句子不改逻辑。本技能的核心哲学是 **「结构优先于措辞」**——先诊断论文类型逻辑、段落职责、声明-证据映射，再逐句润色。具体覆盖：

- **故事架构**：从实验证据中提取核心命题，生成 2-3 个候选故事线，锁定包含 8 个字段的 story package（核心问题、瓶颈、设计原则、主声明、证据映射、机制定位、范围边界、目标期刊画像），防止写作过程中故事漂移。

- **实时期刊情报**：起草前实时搜索目标期刊的 author guidelines（字数上限、摘要格式、图表限额、模板），按 ACS、RSC、Wiley、Nature、Science、Cell Press 六大家族分别适配，不凭记忆猜测。

- **章节 Playbook 起草**：Abstract 六步（背景→缺口→方法→结果→意义→边界）、Introduction 四段漏斗、Results 证据阶梯（合成表征→主性质→基准对比→结构-性能关系→稳定性→推广边界）、Discussion 解读而非复述、Conclusion 三部分收尾。

- **文献引用搜索**：分段文本→解析声明→搜索→5 级保守评估→导出 ENW/RIS/RDF 文件。期刊范围覆盖六大出版家族。

- **质量闸门分级**：Full gate（新起草/大改后）含顶刊表达检索+diction 校准、去 AI 味+中→英修复、逻辑链审计+反向提纲；Light gate（小修）仅后两项。同 session 同主题搜索结果可复用，避免反复搜索。

- **编辑+三轮审稿模拟**：Handling Editor 预审 scope/novelty，A/B/C 三位 reviewer 分域评审（方法证据/领域新颖性/逻辑风格），生成 review_action_matrix。

- **LaTeX 排版诊断**：float 松散、宽扁图、搁浅标题、多面板、SI 密度六类修复，含材料化学领域特有问题（XRD 宽扁、TEM grid、SI 密度）。

- **中→英修复**：8 种中文论文常见模式修复（宽泛重要性先于明确对象、方法列表先于研究缺口、"显著提高"无基线、"首次"无范围等），含材料化学专用的表征归属、性能比较、术语一致性规则。
