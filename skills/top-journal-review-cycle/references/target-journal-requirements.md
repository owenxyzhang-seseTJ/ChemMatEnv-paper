# Target Journal Requirements Protocol

Use this protocol before drafting for a named journal. The requirements packet fields (journal, article type, guideline URL, title/abstract/main-text/display-item/reference limits, required sections, template, etc.) are listed in SKILL.md; this file covers source priority, search queries, and drafting rules — the parts SKILL.md does not.

## Source Priority

1. Official journal author guidelines.
2. Official publisher instructions for authors.
3. Official manuscript templates or article-type pages.
4. Recent accepted articles only as a style reference, not as a replacement for guidelines.

Why this order: guidelines are normative (they state the limits a desk-check enforces); templates are descriptive of format but silent on word limits; accepted articles only show what passed, not what is currently required. Guidelines also change, so a newer official page outranks an older cached one.

## Required Search

Search the web live. Use queries such as:

- `[journal name] author guidelines manuscript format`
- `[journal name] article type word limit abstract`
- `[journal name] manuscript template docx latex`
- `[journal name] graphical abstract TOC graphic requirements`
- `[journal name] reporting summary supplementary information`

## Drafting Rules

- If a requirement is not found, write `not found in checked sources`, not a guess.
- If requirements differ by article type, ask for or infer the article type and mark the inference.
- Treat official current guidance as higher priority than cached local rules.
- Carry the requirements packet into manuscript drafting and review.
- If live search cannot be performed and the user has not provided official guidelines, stop before journal-specific drafting.
- Do not rely on memory for current word limits, title limits, display-item limits, template availability, or reporting requirements.

Why the last two matter: journal limits and reporting requirements drift between years and between article types; a confident-but-wrong word limit surfaces as a desk-reject or a forced last-minute cut, which is costlier than a stop-and-ask.
